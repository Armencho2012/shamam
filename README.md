# Shamam's Sweet Spot

This is the Shamam's Sweet Spot frontend — a Vite + React + TypeScript project using Tailwind CSS and shadcn-ui components.

## Local development

Install dependencies and start the dev server:

```bash
npm install
npm run dev
```

Visit http://localhost:5173 to preview the site.

## Tech stack

- Vite
- React
- TypeScript
- Tailwind CSS
